# Wordlist_Generator :

In This Tool You Can Generate A Big Wordlist By Typing The Information On The Target. You Can Also Make A Basic Word Using This Tool. 


# Tool Installation :

$ apt update 

$ apt upgrade 

$ apt install git

$ git clone https://github.com/CafSecHax/Wordlist_Generator.git

$ cd Wordlist_Generator

$ ls
